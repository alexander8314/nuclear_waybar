# nuclear_waybar
Customise my Waybar on Arch

## Features
- **Reactor meltdown aesthetics**: Glowing blue theme with pulsing effects and heat-warning states when resources spike
- **Live process monitor**: Task manager shwoing top resource-consuming programs with real-time usage bars and percentages
- **Per-process resource graphs**: Visual breakdown of CPU, RAM, and disk usage for each running application
- **Prominent right-side display**: Oversized modular design (150-200px width) for detailed system health monitoring
- **Hyprland integration**: Click-to-kill processes and native workspace switching with reactor-themed status indicators
